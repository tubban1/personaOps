# Interaction Policy & Guardrails

本文件定义了 personaOps 在处理用户互动时的核心准则、安全边界与接管规则。

## 1. 核心优先级
**Brand Constraints (品牌边界) > Persona Style (角色风格)**
任何回复建议必须首先通过品牌合规检查，然后才能应用角色的语气。

## 2. 意图分类与处理矩阵

| 意图 (Intent) | 风险等级 | 处理策略 | 是否转人工 |
| :--- | :--- | :--- | :--- |
| **Praise (夸赞)** | Low | 角色风格回复 | 否 |
| **General Q&A (一般问答)** | Low | 角色语义回复 | 否 |
| **Tour Inquiry (咨询路线)** | Medium | 角色式引导 + 建议转私信 | **是 (建议同步)** |
| **Pricing (咨询价格)** | **High** | 模糊化角色回复 + 记录线索 | **是 (必须)** |
| **Complaint (投诉)** | **High** | 安抚性角色回复 | **是 (必须)** |
| **Harassment (骚扰)** | Critical | 阻断 / 忽略 | 否 |
| **Security/Legal (法律安全)** | Critical | 统一模板回复 / 阻断 | **是 (立即调起)** |

## 3. 人工接管 (Handoff) 触发器

系统在以下情况下必须标记 `requires_human = true`：
1. **转化信号**：提及“多少钱”、“怎么报名”、“有优惠吗”。
2. **信任危机**：提及“骗子”、“退款”、“投诉”、“垃圾”。
3. **情绪激烈**：使用大量感叹号、脏话或极端负面词汇。
4. **决策越权**：涉及价格承诺、合同条款、安全背书。
5. **低置信度**：AI 无法识别意图或识别置信度低于 0.6。

## 4. 角色回复红线 (Taboos)

*   **严禁** 承诺任何未在 `BrandProfile` 中明确标注的价格。
*   **严禁** 替企业做法律或健康相关的担保。
*   **严禁** 与用户进行情绪化对立（即便用户发起攻击）。
*   **严禁** 在回复中引导至非品牌授权的第三方平台。
